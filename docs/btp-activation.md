# BAS-BTP Activation Guide

## 1st Link (Citrix)

1. "Get Started"
2. Input Name -> PH -> Lonza Email -> Lonza User Name (email without "@lonza.com") -> Set Up Password
3. Submit
4. Verify Email

## 2nd Link (Citrix)

1. 1st link sa screen (In my case, agad ako na-redirect sa "Try to create dev space")
2. Lonza email > PW
3. Refresh
4. New Window
5. Copy and Paste the 2nd Link
6. Try to create dev space (SAP Adaptation Project - Cloud Foundry only)
7. Wait until "Starting" turns into "Running"
8. Click your dev space
9. 3 dots on left side
10. Cloud Foundry -> Services -> Log In Required
11. Click: Open a new browser page to generate SSO passcode
12. Click: Identity provider (first option)
13. Log In
14. Accept
15. Copy authenticaion code (from Step 11)
16. Enter your SSO passcode (paste auth code)
17. Sign in
